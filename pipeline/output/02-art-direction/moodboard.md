## Référence 1 : Brasserie Blanc

- **URL** : https://brasserieblanc.com/
- **Secteur** : Brasserie française haut de gamme (UK, chaîne Raymond Blanc)
- **Ce qu'on retient** :
  - Palette bourgogne/crème/céladon — harmonie chaude sophistiquée
  - Typographie Cormorant serif pour les headings — caractère français classique, comparable à Fraunces
  - Photographie lifestyle chaleureuse — lumière naturelle, intérieurs invitants, plats en situation
  - Illustrations custom (escargot, personnages) — touche artisanale personnalisée
  - Hero carousel pleine largeur avec blocs modulaires asymétriques
- **Mesures concrètes** :
  - Taille H1 : ~48-56px
  - Espacement sections : ~100-120px
  - Ratio images : 16:9 (hero), 4:3 (cards)
  - Border radius : 0px (angles droits, sophistication classique)
  - Palette dominante : #8B2332 (bourgogne), #5B7B6B (céladon), #F5F0E8 (crème), #333 (charbon)
- **Cohérence avec notre archétype Caregiver** : L'hospitalité "Maman Blanc" est l'incarnation même du Caregiver — accueil familial, chaleur, soin. La promesse visuelle est celle d'un intérieur soigné.
- **À adapter pour ce projet** :
  - Reprendre la dominante crème chaude en fond principal
  - S'inspirer du contraste serif/sans pour la hiérarchie typographique
  - Simplifier : La Pause est brasserie accessible, pas fine dining — adoucir les angles, ajouter du radius

## Référence 2 : Chez Janou (Paris)

- **URL** : https://www.chezjanou.com/
- **Secteur** : Bistro provençal authentique (Paris, depuis 1996)
- **Ce qu'on retient** :
  - Atmosphère méditerranéenne — tons dorés, bleus, terreux évoquant le sud
  - Hero immersif avec phrase d'accroche émotionnelle ("Un coin de garrigue au cœur de Paris")
  - Design intime et rustique — imagerie naturelle (oliviers), textures organiques
  - Navigation par icônes-boutons de marque — tactile, mobile-friendly
  - Animations fade-in subtiles, storytelling visuel chaleureux
- **Mesures concrètes** :
  - Taille H1 : ~40-48px
  - Espacement sections : ~80-100px
  - Ratio images : 3:2 (ambiance), 1:1 (plats)
  - Border radius : 4-8px (subtil)
  - Palette dominante : dorés chauds, bleu provence, crème, terre
- **Cohérence avec notre archétype Caregiver** : L'intimité et la narration personnelle incarnent la convivialité sincère. L'ancrage provençal (≈ languedocien pour La Pause) crée un pont direct avec la valeur Ancrage local.
- **À adapter pour ce projet** :
  - Reprendre l'approche narrative/émotionnelle du hero
  - S'inspirer de l'ancrage terroir dans les visuels (vins, produits locaux)
  - Augmenter la lisibilité mobile (Chez Janou est un peu dense)

## Référence 3 : Le Comptoir San Rafael

- **URL** : https://www.lecomptoirsr.com
- **Secteur** : Bistrot français / Rôtisserie / Épicerie
- **Ce qu'on retient** :
  - Palette charbon/crème/doré/rouge — classique brasserie française
  - Typographie headline bold et condensée vs. Open Sans body — hiérarchie nette
  - Transitions smooth (0.2-0.6s ease) — mouvement maîtrisé, pas tape-à-l'œil
  - Utilisation de dégradés et ombres subtils pour la profondeur
  - Photography culinaire soignée, emphasis sur le dressage
- **Mesures concrètes** :
  - Taille H1 : ~52-60px
  - Espacement sections : ~100-140px
  - Ratio images : 16:9 (hero), 1:1 (plats)
  - Border radius : 0-4px
  - Palette dominante : #1E1E1E (charbon), #EAEAEA (crème), #FFCB05 (doré), #ED1C24 (rouge)
- **Cohérence avec notre archétype Caregiver** : Le positionnement épicerie/rôtisserie/bistrot multi-format résonne avec la générosité de La Pause (petit-déj au goûter). L'élégance restée accessible incarne le Regular Guy.
- **À adapter pour ce projet** :
  - Remplacer les rouges vifs par notre Terracotta (plus doux, plus Caregiver)
  - Reprendre le principe des transitions 0.3-0.5s ease-out
  - Adapter la largeur contenu pour le mobile-first (80% trafic smartphone)

## Référence 4 : CURATE Annapolis

- **URL** : https://curateannapolis.com/
- **Secteur** : Café artisanal / Boulangerie / Bar à vins
- **Ce qu'on retient** :
  - Esthétique Caregiver raffinée — chaleur + soin + artisanat
  - Photography lifestyle haute qualité — pâtisseries, vins, intérieurs chaleureux

- **Mesures concrètes** :
  - Taille H1 : ~44-52px
  - Espacement sections : ~80-120px
  - Ratio images : 4:3 (galerie), 16:9 (hero)
  - Border radius : 4-8px
  - Palette dominante : noir profond, blanc, #007bff (bleu accent), #61ce70 (vert accent), gris
  - Max-width conteneur : 1170px
  - Breakpoints : 780px / 360px
- **Cohérence avec notre archétype Caregiver** : Le positionnement multi-craft (café + boulangerie + vin) est structurellement identique à La Pause (brasserie + glacier + café). L'attention portée à chaque métier traduit le soin Caregiver.
- **À adapter pour ce projet** :
  - Remplacer le bleu/vert froid par notre palette chaude (Or Miel + Terracotta)
  - Reprendre le whitespace généreux et le max-width ~1170px

## Référence 6 : REIGN Restaurant Toronto

- **URL** : https://www.reigntoronto.ca/
- **Secteur** : Brasserie canadienne moderne (Fairmont Royal York, design Rockwell Group)
- **Ce qu'on retient** :
  - Palette raffinée neutre : crème/off-white #faf8f4, charbon #333, minimalisme
  - Typographie Montserrat (Google Fonts, 100-900) + custom Minerva Modern — hiérarchie élégante
  - Sections split-column alternant texte/image — rythme visuel sophistiqué
  - Hero carousel (6 slides, autoplay 3s, transitions 2s) — immersif mais pas intrusif
  - Design par la retenue et l'élégance — upscale sans excès visuel
- **Mesures concrètes** :
  - Taille H1 : ~56-64px
  - Espacement sections : ~120-160px
  - Ratio images : 16:9 (hero carousel), 3:2 (split sections)
  - Border radius : 0px (angles droits, élégance classique)
  - Palette dominante : #faf8f4 (crème), #333 (charbon), accents minimaux
- **Cohérence avec notre archétype Caregiver** : La retenue visuelle respecte l'intelligence du visiteur — pas de manipulation, pas de surcharge. Le "moins c'est plus" traduit la sincérité.
- **À adapter pour ce projet** :
  - La Pause est plus chaleureuse que REIGN — ajouter davantage de couleur (Or Miel, Terracotta)
  - Reprendre le principe split-column pour les sections (texte gauche / image droite)
  - Réduire l'espacement sections (160px est luxueux, viser 100-120px desktop pour La Pause)
  - Pas de carousel hero (performance mobile, sincérité = montrer directement)

## Synthèse Visuelle

### L'équation La Pause

```
LA PAUSE = Chaleur Méridionale + Respiration Caregiver + Simplicité Regular Guy

Chaleur Méridionale  = Crème/Sable + Or Miel/Terracotta + Fraunces SOFT + photographies terroir
Respiration Caregiver = Whitespace 100-120px + radius 8-16px + fade-in 0.3s + rythme calme
Simplicité Regular Guy = Navigation visible + contenu direct + pas de carousel/parallax + mobile-first 17px+
```

### Mots-clés visuels

- **Chaleur** (Fond crème #faf7f0, accents Or Miel #e4b443, ombres tintées chaudes)
- **Rondeur** (Radius 12px cards, Fraunces SOFT 50+, pas d'angles vifs)
- **Respiration** (100-120px entre sections, 1.6 line-height body, whitespace généreux)
- **Ancrage** (Photographies réelles du lieu, Terracotta languedocienne, vins régionaux)
- **Clarté** (DM Sans 17px+ mobile, hiérarchie nette H1→body, navigation explicite)
